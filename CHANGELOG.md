# Changelog

### Added
- 93 parametric 3D models for DJ and event equipment
- DJ booth components (accordion, apex, flight case styles)
- DJ equipment accessories (wolfmix, cable management, mounts, docks)
- Branded items (keychains, NFC plates, textile logos, coasters)

- Utility functions (SVG import, mesh tools, batch export)

- Reverse engineering workflow documentation

- Contributing guidelines
